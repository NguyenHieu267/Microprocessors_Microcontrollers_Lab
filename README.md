# Microcontrollers - Microprocessors
MCU-MPU Lab for CO3009 course of HCMUT-CSE Semester 251.

## Description 
**Hardware**:
* MCU: STM32F103C6.
* Peripherals: LEDs (single, 7-segment, matrix), buttons/switches, and sensors,.... Depending on the specific lab section
  
**Software** (If you cannot buy STM32F103C6):
* STM32CubeIDE to create .hex file
* Proteus 8 to simulate the circuits

## Requirements
* Lab 1: LED Animations
* Lab 2: Timer Interrupt and LED Scanning
* Lab 3: Buttons/Switches (including FSM and debouncing)
* Lab 4: A Cooperative Scheduler
* Lab 5: Flow and Error Control in Communication (UART)
